Bandwidth utilization report gives the bandwidth utilization of each device. The OIDs required are: 2.2.1.10 (ifInOctets), 2.2.1.16 (ifOutOctets) and 2.2.1.5 (ifSpeed). PolledData for 2.2.1.10 and 2.2.1.16 must store the delta values (instead of absolute counter values) divided by the time period of the delta. Effectively this stores the interface in and out octets in bytes/sec. To achieve this the PolledData for the OIDs must have saveAbsolutes as 'false' and timeAvg as 'true'. PolledData for 2.2.1.5 must store the absolute values (in bits per second), hence saveAbsolutes should be 'true' and 'timeAvg' should be 'false'. The polling period of the 3 PolledData must be equal. Only if a device has all the three OIDs with this specific configuration it will be included in the report. Half-duplex formula is used for utilization computation.
Note:If the above table indicates "0 devices" it means that no data was collected for any of the devices which matched the required criteria.
